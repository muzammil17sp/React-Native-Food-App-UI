This is a code repository for the Food App  



- React Native(expo)
- Navigation
- Reducer and Context Api
- basket(cart) functionality
- Swipe List 
- Custom Font



# Installation Steps

## Using npm

Run commands

1) ```npm install```


2) ```npm start```


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn start```
